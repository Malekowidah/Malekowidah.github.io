What I built

Graduation Project

Graduation Project

BorderEye

A real-time simulation system designed to detect and track suspicious objects in a virtual border environment — combining network engineering with intelligent object recognition logic.

  • Assigns unique IDs, detects object shape, and tracks speed & acceleration of unverified entities in real time.
  • Differentiates between suspicious and authorized objects — only reacts to those that fail to send valid acknowledgment signals.
  • Models networked components and visualizes object tracking within a full simulation environment.
